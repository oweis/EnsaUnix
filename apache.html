Apache

Le logiciel libre Apache HTTP Server (Apache) est un serveur http créé et maintenu au sein de la fondation Apache C'est le serveur HTTP le plus populaire du Word Wide Web. Il est distribué selon les termes de la licence Apache.


Instalation :

apt-get install apache2


Pour être sure que l'apache est bien installer , verifier si ce message s'affiche :


Apres cette installation votre serveur doit fonctionner et être accessible à l'adresse suivante http://localhost (à partir de la même machine) ou ton addresse ip.
* les fichiers de configuration de Apache2 sont dans le dossier /etc/apache2.
* Utilisation sudo service apache2 start | stop | restart | status.
