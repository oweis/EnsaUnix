Instalation :

Il est preferable de lancer apt-get update avant (aide a minimiser les erreurs)



Puis on installe les paquets slapd et ldap-utils :
apt-get install slapd
apt-get install ldap-utils

Pendant l'installation, un mot de passe est demandé pour le "rootDN". Cet utilisateur dispose de tous les droits sur l'arbre de données, et uniquement sur l'arbre de données, puisqu'il n'existe pas dans l'arbre de configuration.







On saisit dans le terminal la commande suivante:
dpkg-reconfigure slapd

sudo ldapsearch -Q -LLL -Y EXTERNAL -H ldapi:/// -b cn=config dn
