
EnsaUnix
=======
A static website to teach beginners the configuration and use of some UNIX services(Apache,MySql,DNS,Samba,NFS,SSH,Netfilter,WebDav,LDAP,SendMail)

Getting Started
-------
Each service contain 3 pages :
=> Page 1 : How to install/configure the service,
=> Page 2 : Show the tests results,
=> Page 3 : Script Shell.
